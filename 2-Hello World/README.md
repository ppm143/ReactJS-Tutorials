> ## Installation

- Node JS Stable Version

> ## Create React App

1. ### **Using npx**

- **npx is npm package runner**
- `npx create-react-app project-name`
- `cd project-name`
- `npm start`

2. ### **Using npm**

- `npm install create-react-app -g`
- `create-react-app project-name`
- `cd project-name`
- `npm start`

**Use npx mostly**
